第2章 一切都是对象

  • 2.1 句柄：即引用，或C++里的指针，我们可以用它来操纵对象（句柄与对象之间的连接可以看成是遥控器和电视机之间的关系）
  • 2.2.2 主要类型 ：如boolean封装成Boolean，char封装成Character， int封装成Integer
  • 2.2.3 Java的数组：在C++里面， 尽量不要使用数组（只是为了防止越界？），换用STL里更安全的容器
  • 2.4.1 初始化：在类的成员会自动初始化，即创建一个类的对象，只需要new，而不需要对每一个字段赋值；但是局部变量不会被初始化，要手动编写
